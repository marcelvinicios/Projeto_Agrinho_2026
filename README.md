# 🌾 EcoCultivar - Agricultura Sustentável

Uma página web institucional e interativa dedicada à conscientização e educação sobre os pilares da **Agricultura Sustentável**. O projeto apresenta conceitos fundamentais, práticas ecológicas do setor e conta com um desafio dinâmico para testar os conhecimentos do usuário.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais, mantendo uma estrutura limpa, sem dependências externas e dividida de forma semântica:

*   **HTML5:** Estruturação semântica do conteúdo (`<header>`, `<section>`, `<nav>`, `<footer>`).
*   **CSS3:** Design moderno, layout responsivo com *Flexbox* e *CSS Grid*, e uso de variáveis CSS para padronização de cores.
*   **JavaScript (Vanilla):** Lógica nativa para interatividade da seção de testes (quiz) com feedbacks em tempo real.

---

## 📦 Estrutura do Projeto

A arquitetura do código foi dividida em três arquivos principais para garantir a organização e facilidade de manutenção:
